A base do React - Componentes de função vs de classe
Nesta aula, exploraremos mais especificamente a diferença entre componentes de função e componentes de classe.

Primeiramente, você aprenderá sobre os componentes de função, que são uma forma mais simples e concisa de criar componentes no React. Também entenderá como definir um componente de função.

Em seguida, abordaremos os componentes de classe, que são uma abordagem mais antiga para a criação de componentes no React. Você aprenderá como definir um componente de classe, como trabalhar com o ciclo de vida do componente e como lidar com estados.

Componentes no React são pequenos pedaços. https://pt-br.legacy.reactjs.org/docs/components-and-props.html#gatsby-focus-wrapper

A base do React - Props: passando propriedades para os componentes
Nesta aula, você aprenderá como passar propriedades para os componentes no React. As props são utilizadas para enviar dados de um componente pai para um componente filho, permitindo a personalização e reutilização dos componentes. Durante esta aula, você entenderá como definir e utilizar props, como passá-las entre componentes e como acessar seus valores dentro dos componentes. Compreender o conceito de props é fundamental para o desenvolvimento de aplicações React flexíveis e modulares.

Atenção: Quando usamos o vite, a extensão dos arquivos sempre serão .jsx. Para rodar o projeto apenas use npm install e em seguinda npm run dev
