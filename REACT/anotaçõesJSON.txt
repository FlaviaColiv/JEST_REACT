A base do React - Introdução ao React.js
Nesta aula, você será introduzido ao React.js, uma das bibliotecas mais populares para a construção de interfaces de usuário em aplicações web. Durante a aula, você aprenderá os conceitos fundamentais do React e como começar a utilizá-lo.

Você começará entendendo o que é o React e por que ele se tornou tão popular entre os desenvolvedores web.

React é uma biblioteca que serve para a criação de interfaces de usuário.

A base do React - O JSX
Nesta aula introdutória, você será apresentado ao conceito de JSX e sua importância no desenvolvimento de aplicações React. O JSX é uma extensão da sintaxe do JavaScript que permite escrever código HTML/XML dentro do JavaScript.
