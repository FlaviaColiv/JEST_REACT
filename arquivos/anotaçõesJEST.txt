Testes automatizados de software com Jest - Básico - Quais tipos de teste existem?
Nesta aula sobre testes automatizados de software com Jest, você aprenderá sobre os três principais tipos de testes: testes unitários, testes de integração e testes de ponta a ponta. Os testes unitários verificam unidades isoladas do código, os testes de integração testam a interação entre diferentes unidades e os testes de ponta a ponta simulam a interação do usuário com a aplicação. O Jest será utilizado para escrever e executar esses testes, garantindo a qualidade do software.

Testes automatizados de software com Jest - Básico - O que são test runners?
Nesta aula sobre testes automatizados de software com Jest, você aprenderá sobre test runners, que são ferramentas responsáveis por executar os testes automatizados de um projeto. Você entenderá o papel dos test runners e aprenderá como configurar e utilizar o Jest como test runner.
