TypeScript - Entendendo o TypeScript

TypeScript é uma linguagem de programação fortemente tipada que se baseia em JavaScript, o nome dela já diz Type, de tipo, então ela vai ser muito usada quando queremos tipar nosso código, calma que logo vamos explicar por que tipagem é tão importante.
O TS foi desenvolvido pela Microsoft e é considerado um superset do JS, que seria uma linguagem construída “em cima” do javascript tradicional.

TypeScript - Instalação e primeiros passos de uso

Vamos começar a entender como instalar e compilar o typescript. É importante começar dizendo que o código em ts é compilado para js.
Compilar seria pegar um código em um determinado formato e transformar pra outro.

Comandos utilizados na aula
npm init -y - inicializa o npm no projeto sem perguntar as informações
npm install -g typescript - instala o TypeScript globalmente no PC OU
npm install --save-dev typescript - instala somente na pasta (ajuda na hora clonar, pois só atualiza, aí usa
npm install (puro) que já roda)
npx tsc exemplo-compilacao/index.ts - executa o compilador do TypeScript para transformar o arquivo .ts para .js
npx tsc --init - cria o arquivo de configuração do TypeScript no projeto

Entenda os tipos no TypeScript - Parte 1 ✔️

TypeScript - Entenda os tipos no TypeScript - Parte 1

Os tipos, aqui começa uma das maiores diferenças do ts para o js, a tipagem. O TypeScript oferece todos os recursos do JavaScript e uma camada adicional em cima deles: o sistema de tipos do TypeScript. O JavaScript fornece tipos primitivos de linguagem como string e número, mas não verifica se atribuímos eles de forma correta. O TypeScript sim vai fazer essa verificação.

Sendo mais produtivo com a verificação de tipo estático ✔️

TypeScript - Sendo mais produtivo com a verificação de tipo estático

O grande objetivo do TypeScript é ser um verificador de tipo estático para programas JavaScript - em outras palavras, o sistema interpreta o código antes dele ser executado e garante que os tipos do código estejam corretos. Então, ele nos ajuda a identificar possíveis erros ou problemas no nosso código conforme escrevemos no VS Code, então antes mesmo de executar esse código.

Tipos Tuples: uma coleção mais específica que o Array ✔️

TypeScript - Tipos Tuples: uma coleção mais específica que o Array

Tuples, ou Tuplas como chamamos em português não é um conceito novo na programação Um tipo de Tuple é outro tipo de tipo Array mas a diferença é que ele sabe exatamente quantos elementos esse array contêm e exatamente quais tipos de dados ele contêm em posições específicas.
