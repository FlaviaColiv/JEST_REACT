Entendendo o TypeScript ✔️

TypeScript - Entendendo o TypeScript

TypeScript é uma linguagem de programação fortemente tipada que se baseia em JavaScript, o nome dela já diz Type, de tipo, então ela vai ser muito usada quando queremos tipar nosso código, calma que logo vamos explicar por que tipagem é tão importante.
O TS foi desenvolvido pela Microsoft e é considerado um superset do JS, que seria uma linguagem construída “em cima” do javascript tradicional.

Instalação e primeiros passos de uso ✔️

TypeScript - Instalação e primeiros passos de uso

Vamos começar a entender como instalar e compilar o typescript. É importante começar dizendo que o código em ts é compilado para js.
Compilar seria pegar um código em um determinado formato e transformar pra outro.
Links de apoio:
Download do Node.js
Documentação do TsConfig

Comandos utilizados na aula
npm init -y - inicializa o npm no projeto sem perguntar as informações
npm install -g typescript - instala o TypeScript globalmente no PC
npx tsc exemplo-compilacao/index.ts - executa o compilador do TypeScript para transformar o arquivo .ts para .js
npx tsc --init - cria o arquivo de configuração do TypeScript no projeto

Código realizado durante a aula
function sendMessage(userName: string, isLogged: boolean) { console.log(`Olá ${userName}. Você ${!isLogged? " não" : ""} pode logar.`) }
const userName = "Beto" const isLogged = true
sendMessage(userName, isLogged)
