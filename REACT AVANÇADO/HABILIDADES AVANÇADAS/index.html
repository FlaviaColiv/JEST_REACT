Custom Hooks com React

Habilidades Avançadas - Custom hooks no React

Nesta aula, exploramos o conceito de hooks personalizados no React, uma técnica avançada para reutilizar a lógica de estado e efeitos em seus componentes React. A aula cobre os fundamentos de como criar e usar hooks personalizados, proporcionando uma base sólida para desenvolvedores que querem aprimorar suas habilidades no React.

Criando um custom hook no React

Habilidades Avançadas - Criando um custom hook no React pratica

Esta aula é um mergulho profundo na criação de um custom hook no React. Abordamos passo a passo o processo de desenvolvimento de um hook personalizado, desde a identificação de um caso de uso comum até a implementação e teste. Ideal para desenvolvedores que já têm conhecimento básico em hooks e querem expandir sua caixa de ferramentas de desenvolvimento.

Como nomear seus custom hooks no React - Parte 1

Habilidades Avançadas - Como nomear seus custom hooks no React

Nesta aula, focamos em uma parte crucial, porém muitas vezes negligenciada, do desenvolvimento de hooks personalizados: a nomeação eficaz. Discutimos as melhores práticas para nomear seus hooks de maneira clara e intuitiva, garantindo que eles sejam facilmente compreendidos e utilizados por outros desenvolvedores no projeto.

Sempre começa com "use" para nomear.

Habilidades Avançadas - Criando um custom hook no React - Parte 2
