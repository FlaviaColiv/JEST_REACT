Usando states em um class component

React Avançado - Usando states em um class component Nesta aula de React Avançado, você aprenderá sobre o gerenciamento de estado utilizando componentes de classe. O React oferece diferentes abordagens para lidar com o estado de um componente, e nesta aula, focaremos na utilização de class components. Primeiramente, você entenderá o conceito de estado em um componente React e sua importância no desenvolvimento de aplicações. Aprenderá como criar e inicializar o estado em um class component e como atualizá-lo de forma apropriada. Em seguida, exploraremos as diferentes formas de manipular o estado em componentes de classe. Você aprenderá a atualizar o estado utilizando o método setState, entendendo como fazer alterações incrementais, como lidar com objetos complexos e como trabalhar com estados dependentes do estado anterior. Componente Stateless vs Componente Statefull

React Avançado - Componente Stateless vs Componente Statefull Nesta aula sobre React Avançado, você aprenderá sobre a diferença entre Componente Stateless (sem estado) e Componente Stateful (com estado). Será explicado o conceito de Componente Stateless, que é um componente React que não possui um estado interno. Esses componentes são mais simples e focados na apresentação de informações, sendo úteis para renderizar dados estáticos ou receber props e exibir conteúdo na interface. Por outro lado, você também aprenderá sobre Componentes Stateful, que têm um estado interno e podem lidar com alterações dinâmicas ao longo do tempo. Esses componentes são responsáveis por gerenciar dados e responder a eventos, permitindo a interatividade na aplicação React. Serão abordados conceitos como a criação de um estado, a atualização do estado e a renderização condicional com base no estado atual.

Lifecycle ou ciclos de vida no React

React Avançado - Lifecycle ou ciclos de vida no React Nesta aula, você aprenderá sobre os ciclos de vida ou "lifecycle" no React. Os ciclos de vida são métodos especiais que são executados em momentos específicos durante o ciclo de vida de um componente React. Você explorará os diferentes estágios do ciclo de vida de um componente, desde a sua criação até a sua remoção. Aprenderá sobre os métodos que são chamados em cada estágio, como componentDidMount, componentDidUpdate e componentWillUnmount, entre outros. Introdução aos Hooks

React Avançado - Introdução aos Hooks Nesta aula introdutória, você será introduzido aos Hooks, uma poderosa funcionalidade do React que permite a você utilizar estado e outras características do React em componentes funcionais. Primeiramente, você aprenderá o que são Hooks e por que eles foram introduzidos no React. Os Hooks são uma forma de reutilizar lógica de estado e efeitos colaterais em componentes funcionais, eliminando a necessidade de escrever componentes de classe. Você entenderá as vantagens dos Hooks em relação aos componentes de classe tradicionais. Link de apoio: Built-in React Hooks – React

Se você está familiarizado com os métodos do ciclo de vida do React, você pode pensar no Hook userEffect como componentDidMount, componentDidUpdate e componentWillUnmount combinados.

Regras:
- Não chame Hooks dentro de loops, condições ou funções aninhadas.
- Apenas chame Hook de componentes funcionais. Não chame Hooks de funçoes JS.
